ncre4 网络工程师 day1

操作系统

  • 操作系统构造三大方法 ： 整体结构、层次结构、微内核C/S结构

  • 批处理系统的优点 作业流程自动化高，资源利用率高 作业吞吐量大 缺点 缺少交互性（批处理不就是交互性差么~）

  • 操作系统内核态下运行 -》 屏蔽中断 内核态即管态

  • 操作系统提供给用户用于应用程序编程的唯一接口是 系统调用

  • 与进程具有一一对应关系的是PCB 进程控制块 （包含进程标识符 进程当前状态 代码起始指针）

  • 进程中有线程被调用投入运行，进程运行状态是 运行状态

  • Pthread线程包中

  • 一定不会引起进程调度的：一个进程从就绪状态变成运行状态

  • P：申请资源 V：释放资源 信号量mutex初值为1 当K个进程在Mutex的等待队列，信号量值为 - k

  • 允许动态扩充内存容量的方案：虚拟页式

  • 可变分区回收 S:起始地址 L：长度 上邻分区空闲 登记项地址 = S - L 下邻分区空间 S + L

  • 页式存储管理方案 虚拟地址空间2GB 页面大小4KB 4字节表示物理页号 2048 / 4 = 512

  • FIFO 页面置换方法有可能产生Belady异常现象

  • LRU 页面置换 最近最少使用：缺页时，置换掉最长时间未被使用的页面

  • LFU 最近不常用： 置换掉被访问次数最少的页面

  • OPT 理想页面置换方法 置换以后不再需要或者最长时间采用的页面

  • NRU 从类编号中最小的非空类中选出一个页面置换掉

  • 虚拟存储管理方案 抖动 是页面置换算法不合理导致

  • 文件存取方式依赖于 文件的存储结构 存放设备的物理特性

  • 文件系统实现文件按名存取是通过 文件目录查找工作完成的

  • 打开文件系统调用后 系统会返回fd 文件描述符

  • 无分组操作磁盘空间利用率 = 每个逻辑记录长度 / 块大小

  • 分组操作利用率 = 块因子 * 每个逻辑记录长度 / 块大小

  • 磁盘读写性能 寻道时间main 传输时间 旋转延迟

  • 死锁产生原因 资源分配不当 进程推进顺序不当

  • 死锁状态一定是不安全状态 安全状态不可能进入死锁状态

  • 最多申请资源数 = ceil(资源数 / 进程数)

  • 存储设备： 寄存器 高速缓存 内存 硬盘

  • 进程从运行状态变为就绪状态： 被调度程序抢占处理机 进程创建完成 时间片用完

  • 第一套多选4 很重要 看一下！

  • 地址映射通常是硬件完成的 将虚拟地址转化为物理地址

  • 块表 存在于高速缓存之中 TLB 切换进程需要刷新块表 对块表的查找内容是并行的

  • FCB 文件控制块 文件名 文件大小 文件创建时间 磁盘起始地址 FD是文件打开之后生成的

  • 提升文件系统性能： 块高速缓存 磁盘空间合理分配 磁盘驱动调度

  • 设备和CPU数据传送方式 程序直接控制 中断 DMA 通道控制

  • 解除死锁的方法 剥夺 撤销 重启

NET 网络

  • 对计算机发展重要影响的广域网 ARPANET

  • TCP 传输控制协议 面向连接 UDP 用户数据报协议 面向非链接

  • 网络协议由语法 语义 时序 三要素组成 组织方式为分层 语法是对事件的时序顺序说明

  • OSI参考模型各层功能

  • 共享介质局域网 采用一个发送 其他收听 广播数据帧的方式 不可避免冲突 总线型

  • 以太网长度字段值不包含前导码 帧前定界符

  • MTU 一个帧最多能携带的数据量

  • IPV6 书写规范

  • CS模型中 服务器相应客户请求有两种方案 并发服务器方案 重复服务器方案

  • pop3 中 PASS 用户邮箱口令 RETR 请求服务器发送制定编号邮件 STAT 查询报文总数和长度

  • 即时通讯最早是以色列 Mirabils 的ICQ 使用P2P 并非CS模式 RFC2778规定了通讯模型

  • SIP工作在应用层 基于SIMPLE XMPP基于JABBER

  • netbios最早在Windows使用 名称可以由19个英文字符组成 限制254个线路

  • NES 采用CS模式

  • 全文搜索引擎模块组成 搜索器 检索器 用户接口 索引器

  • X.800 由ITU-T指定 分为被动攻击和主动攻心 提供5类14个安全服务

  • BlowFish 是BS设计的对称分组码 对称加密 分组长度64B 迭代16轮

  • kerberos协议 MIT开发 包含CS 可以产生回话秘钥 基于对称秘钥

  • 可信任计算机评估 A -> D 3 -> 1 五级 -> 一级

  • TCP由kahn指定 最底层是主机-网络层 OSI由ISO指定

  • 属于无线网络的 WSN WMN ADHOC WLAN BT

  • 线路类型：

  • RIP协议解决缓慢瘦脸 可以采取 水平分割 毒性逆转 出发刷新 限制最大距离

  • IP服务特点 不可靠 无连接 尽最大努力

  • IP电话中 网关的功能 号码查询 信号调制 路由控制
